#№ Кейс 2: Интерактивная система визуализации технологического процесса
### Обучающий симулятор "Электролиз 360" для практикантов РУСАЛ

| Раздел        | Описание |
|---------------|----------|
| **Цель**      | Создать безопасный, интерактивный симулятор, который сократит время на первичное обучение практикантов и повысит их понимание технологического процесса. |
| **Описание**  | Разработка веб-приложения, симулирующего процесс электролиза алюминия. Система должна позволять практикантам изменять технологические параметры (сила тока, температура) и наблюдать за их влиянием на ключевые показатели эффективности (выход по току, расход энергии) в реальном времени. |
| **Проблематика** | 1. **Сложность обучения:** Процесс электролиза сложен и опасен для экспериментов.<br>2. **Отсутствие наглядности:** Практикантам сложно понять взаимосвязь между параметрами и результатом.<br>3. **Необходимость стандартизации:** Требуется единый, интерактивный инструмент для обучения. |
